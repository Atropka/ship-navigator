# ship-navigator

# Запуск приложения

## Frontend

```
cd Frontend
npm install
npm run dev
```

## Backend

```
cd Backend
pip install djangorestframework, corsheaders, 
python manage.py runserver
```

> Для работы с панелью администратора перейдите на http://127.0.0.1:8000/admin 

> Данные для входа: admin, 1
